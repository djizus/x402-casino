Pixi Poker Table Asset Requirements
===================================

1. room_bg.png
   - Type: Top-down 2.5D painted room background (floor, subtle walls/fade)
   - Size: 2048x2048 PNG, alpha background so table can sit above
   - Features: Radial lighting vignette, soft reflections hinting at casino floor tiles, no furniture.

2. table_base.png
   - Type: Oval poker table seen from ~30° top-down
   - Size: 1600x900 PNG with transparent background
   - Details: Green felt center, dark mahogany padded rail, gold betting line. Include shadow baked around edges.

3. table_highlight.png
   - Type: Additive glow overlay for table rim
   - Size: 1600x900 PNG
   - Details: Soft yellow/orange gradient ring matching table_base size for spotlight effects.

4. seat_empty.png
   - Type: Single chair viewed from same camera angle
   - Size: 320x420 PNG
   - Details: Leather casino chair with subtle drop shadow, centered pivot at seat center.

5. seat_occupied_overlay.png
   - Type: Glow overlay to drop atop seat when a player is present/active
   - Size: 320x420 PNG, mostly transparent.

6. avatar_robot_sheet.png
   - Type: Sprite sheet of 6 bust icons for different robot avatars
   - Size: 6 columns x 1 row, each cell 256x256 (total 1536x256)
   - Style: Futuristic robot busts with varied colors, friendly expressions, top-lit to match table.

7. card_faces_sheet.png
   - Type: Sprite sheet containing all 52 poker card faces plus one blank slot
   - Layout: 13 columns x 4 rows, each cell 180x260 px, transparent corners
   - Style: Classic casino face design, crisp typography.

8. card_back.png
   - Type: Single card back texture
   - Size: 180x260 PNG
   - Details: Blue geometric pattern with gold trim.

9. chip_stack_red.png / chip_stack_blue.png / chip_stack_green.png
   - Type: Three PNGs, each 220x220, showing small stacks of colored chips with shading.
   - Include slight perspective to emphasize 2.5D feel.

10. chip_single_white.png
    - Type: Single chip sprite
    - Size: 128x128 PNG
    - Use for flying animations.

11. dealer_button.png
    - Type: Top-down dealer puck, ivory with “DEALER” text
    - Size: 160x160 PNG

12. pot_glow.png
    - Type: Additive-only radial glow for pot area
    - Size: 512x512 PNG with soft orange/yellow gradient

13. action_banner.png
    - Type: Wide UI tag to display last action per seat
    - Size: 420x140 PNG with transparent center
    - Style: Dark slate background, gold trim, subtle bevel.

14. badge_small.png
    - Type: Circular badge used for seat numbers/highlights
    - Size: 128x128 PNG, neutral grey, white stroke.

15. particle_dot.png
    - Type: Tiny blurred dot for confetti/win effects
    - Size: 64x64 PNG, pure white soft circle.

Nano Banana Prompt
------------------
“Create a cohesive set of high-resolution PNG sprites for a top-down 2.5D futuristic-casino poker scene. Assets needed: (1) 2048x2048 room floor background with warm spotlight vignette, (2) 1600x900 oval poker table with green felt, mahogany rail, subtle shading, plus matching glow overlay, (3) single leather casino chair sprite viewed from slight top angle and an additive glow overlay for occupied seats, (4) sprite sheet of six colorful friendly robot bust avatars (256x256 cells, 6 columns), (5) complete poker card face sheet (13x4 grid, 180x260 each) plus separate ornate blue card back, (6) stacks of red, blue, and green chips plus a single white chip sprite, (7) ivory dealer button, (8) soft orange pot glow, (9) action banner plaque, (10) small circular badge, (11) blurred white particle dot. Style: polished AAA game UI, soft global illumination, slight perspective to feel 2.5D, transparent backgrounds, consistent lighting from top center.”
